# Bug Seer

[![ES](https://img.shields.io/badge/ES-Español-red)](README.es.md)

Habilidad Bug Seer para OpenClaw.

##Qué Soluciona
Predice zonas problemáticas de errores usando patrones de código y datos históricos de defectos.

## Características
- Activación automática cuando se detectan tareas relevantes
- Resultados profesionales listos para producción
- Enfoque de seguridad primero
- Soporte de reversión

## Uso
`/bug-seer`

## Ejemplo
```
Analiza el código en busca de problemas de análisis.
```

## Instalación
Esta habilidad está automáticamente disponible en OpenClaw.
